# FizzBuzz

Print the numbers from 1 to 100.

###Rules

- For multiples of three print Fizz instead of the number
- For multiples of five print Buzz
- For numbers which are multiples of both three and five print FizzBuzz

### Sample-Output

```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
17
Fizz
19
Buzz
... etc up to 100
```
